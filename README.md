# StaticWorld

Lock down your world to certain chunks so you save memory

Commands:
- /staticworld allowchunks - Put chunks in your selection to the whitelist (SimpleWorldEdit required)
- /staticworld denychunks - Removes chunks in your selection from the whitelist (SimpleWorldEdit required)
- /staticworld enable - Chunks not in the whitelist will now be blocked
- /staticworld disable - All chunks can be requested again

SimpleWorldEdit is ONLY required for the setup. Once your regions are set, you can uninstall it.
